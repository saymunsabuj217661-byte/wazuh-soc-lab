# Wazuh SOC Lab

## Project Overview

This project demonstrates the implementation of a Security Operations Center (SOC) lab using Wazuh SIEM and a centralized Rsyslog server.

## Objectives

- Build a centralized log management solution
- Deploy Wazuh SIEM
- Monitor Windows and Linux endpoints
- Integrate router syslog
- Practice security monitoring and alert investigation

## Lab Components

- Ubuntu Server (Rsyslog + Wazuh)
- Windows Agents
- Ubuntu Agents
- Router Syslog
